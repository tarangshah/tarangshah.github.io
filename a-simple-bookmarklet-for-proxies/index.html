A simple bookmarklet for using a proxy

11 Jan 2015

This is a simple bookmarklet for visiting sites blocked by your ISP.

First create a bookmark in you bookmarks bar.

Right click on it and click on "Edit...". Firefox users must click on "Properties" here.

Then in the dialog which pops up, edit the URL or Location Field. Replace everything in the URL or Location text box with the code given below.

javascript:(function(){document.location.href="http://mirrorrr.appspot.com/"+document.URL;})();

Then whenever you encounter a blocked site, just click on the button and you'll be redirected to the site through a proxy site.

Do comment if you need any help

The proxy site URL can be edited to suit your needs. The only requirement is that the proxy should work with the "proxysite.com/anythinghere/<URL>" format.

If thats the case, then just replace the default proxy URL in the bookmarklet and you're set.
